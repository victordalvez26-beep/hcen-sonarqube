Docs para HCEN (central)

Contenido:

- `openapi-hcen.yaml` — OpenAPI 3.0.3 de la API central HCEN para gestionar nodos (endpoints /nodos, /nodos/{rut}/config, /nodos/{rut}/notify, etc.).

Cómo usar:

- Usar Swagger UI, Redoc o cualquier visualizador OpenAPI para abrir `openapi-hcen.yaml`.
- Las rutas públicas usan `RUT` como clave lógica (path parameter). El `id` en los esquemas es un identificador interno y no debe usarse desde el front-end público.

Notas:
