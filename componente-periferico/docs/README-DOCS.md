Docs para el nodo periférico

Contenido:

- `openapi-periferico.yaml` — OpenAPI 3.0.3 de la API del nodo periférico (endpoints de prueba como /auth, /nodos/{rut}).

Cómo usar:

- Para visualizar localmente con Swagger UI o Redoc, copia `openapi-periferico.yaml` al visualizador o usa la URL local si sirves estático el archivo.
- Ejemplos en el YAML usan RUT como clave lógica. El `id` es un identificador interno y no debe usarse desde la UI pública.

Notas:
- Este README es complementario al README principal del proyecto. Incluye solo los recursos de documentación de la API.
